I am SlayingMantis and my coding journey began in January 2025 when I came across Exercism.org's Coding Fundamentals Bootcamp. I decided to take a leap of faith and enroll, unsure if I was going to be stuck in another beginner tutorial hell.

It turned to be a wonderfully-crafted learning experience with a vibrant community that was more than happy to welcome absolute beginners like myself. Within a few weeks, I realized I was having so much fun learning the fundamentals of coding, I became addicted. The rest is history.

Below are some of the games that I learned to develop through my journey in the bootcamp. So far, I've got one game, Breakout, running upto a reasonable level of functionality. I aim to add further features, like Highscores, advanced levels, etc., as I grow in my programming capabilities. More games, like Snake, will also be added as time permits.

Hope you have fun playing! I had a blast making them!
